# 🐹 Golang applications

Small Golang projects i make to explore this language.

## 🛠️ Local Development

Interested in the code? Wanna run it locally?

**Prerequisites:**

- [Go](https://golang.org/dl/) (v1.20 or higher) installed on your machine.

1.  **Clone the repo:**

    ```bash
    git clone https://github.com/yurirodnov/touching-go
    ```

    or via SSH

    ```bash
    git clone https://github.com/yurirodnov/touching-go
    ```

2.  **Go to a specific directory:**
    ```bash
    cd 001_health_checker (or any another folder)
    ```
3.  **Compile from source:**

    ```bash
    go build .
    ```

4.  **Run compiled program:**

    **Windows:**

    ```bash
    .\health_checker.exe
    ```

    **Linux / macOS:**

    ```bash
    ./health_checker
    ```
